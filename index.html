Proqramlaşdırma

Proqramlaşdırma Dilləri

HTML

Bütün veb-qovşaqlarda toplanmış və internetə qoşulmuş kompüterlərin ekranlarında göstərilən sənədlər HTML proqram kodunda yazılırlar. HTML – səhifələrdə mətn bloklarının, təsvirlərin yerləşdirilməsinə, cədvəllərin qurulmasına, sənədin və sənəddəki mətnin rənglərinin seçilməsinə, multimediya elementlərinin əlavə edilməsinə, hiperistinadların və bütün bu elementlər arasında əlaqələrin yaradılmasına imkan verir.

CSS

Veb səhifələr üçün universal şablonlar hazırlamağa imkan verir və rəng, şrift, ölçü kimi hərflərin tərzini müstəqil şəkildə dəyişdirmək üçün də istifadə edilə bilər. Bu texnikanın ən vacib xüsusiyyəti istifadəsində rahatlıqdır.

JAVASCRIPT

JavaScript, veb brauzerlərdə çox istifadə olunan dinamik bir proqramlaşdırma dilidir. JavaScript-də yazılmış müştəri ssenariləri sayəsində istifadəçi ilə qarşılıqlı əlaqə, brauzerə nəzarət, serverlə asenkron əlaqə və veb səhifə məzmununun dəyişdirilməsi kimi funksiyalar təmin olunur. JavaScript də Node.js. kimi platformalar sayəsində server tərəfində geniş istifadə olunur.

BOOTSTRAP

veb səhifələri və tətbiqetmələri yaratmaq üçün istifadə olunan açıq mənbə alətidir və HTML və CSS vasitəsiylə yazı növləri, formlar, düymələr, navigasiya və başqa komponentlər üçün istifadə olunan xüsusi dizayn qəliblərindən ibarətdir. Əsas hədəf dinamik veb səhifələri, veb tətbiqetmələri yaratmaq və onların inkişafını sadələşdirməkdir.

PHP

dinamik veb səhifələr yaratmaq üçün nəzərdə tutulmuş bir skriptləşdirmə dilidir.PHP Əsasən veb həllərin yaradılmasında istifadə edilən populyar server proqramlaşdırma dilidir. Məsələn, formadakı məlumatları qəbul edərək emal edən skript yaratmaq, saytda axtarış sistemi reallaşdırmaq və ya qeydiyyat sistemi yaratmaq mümkündür.

LARAVEL

MVC quruluşunda veb tətbiqetmələrin inkişafı üçün hazırlanmış pulsuz, açıq mənbəli PHP veb tətbiqetmə çərçivəsi. Laravel, MIT lisenziyası altında, GitHub saytında yerləşdirilən mənbə kodu ilə sərbəst buraxılmışdır.
